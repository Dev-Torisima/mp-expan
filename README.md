# MinusP（mp言語）拡張サポート  

mp言語は十分な学習なしで日本語でもプログラミングができる言語です  
この拡張は以下をサポートします  
・シンタックスハイライト（色付け）  
・スニペット  
・キーワードの補完／定数や関数名の補完  
.cmppまたは.cmph拡張子がターゲットです  

対応しているのは以下のバージョンです
・Windows11
・それ以前の64bit版
（注意：Web版には対応していません）

-p（mp言語）仕様はhttps://torisima.f5.si/mp/ex/から
  
  
# MinusP(mp-lang) extended support  

mp language is programming language without learning well.  
This supports the following.  
- Syntax highlighting  
- Snippets  
- Keyword completion  
- Built‑in constants and functions  
This targets ".cmpp" or ".cmph".  

The supported versions are as follows:
- Windows 11
- Earlier 64-bit versions
(Note: The web version is not supported)
