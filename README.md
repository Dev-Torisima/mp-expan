# mp言語（-p）拡張サポート  

mp言語は十分な学習なしで日本語でもプログラミングができる言語です  
この拡張は以下をサポートします  
・エラー判定（ウェブ版を除く）  
・シンタックスハイライト（色付け）  
・スニペット  
・キーワードの補完／定数や関数名の補完  
.cmppまたは.cmph拡張子がターゲットです  

対応しているのは以下のバージョンです  
・Windows11  
・それ以前の64bit版  
・ウェブ 

mp言語の仕様は https://torisima.f5.si/mp/ex/ からご確認ください  
  
  
# mp-lang(-p) extended support  

mp language is programming language without learning well.  
This supports the following.  
- Error detection (exclude web version)  
- Syntax highlighting  
- Snippets  
- Keyword completion  
- Built‑in constants and functions  
This targets ".cmpp" or ".cmph".  

The supported versions are as follows:  
- Windows 11  
- Earlier 64-bit versions  
- Web
